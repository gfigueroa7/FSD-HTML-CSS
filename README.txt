------- PROYECTO -------

La idea del proyecto es una pagina donde puedas encontrar varias cosas para tu mascota

Que este todo concentrado en un solo lugar para facilitar a los usuarios

Y si recien empezas poder contar con un lugar donde este todo y no tengas que buscar en google todo por separado

Aparte la gente va a valorar y asi tener un mejor feedback

Cualquiera se puede postular eso va a ser gratuito y asi salir en la web

Va a ver "pagos" para apoyar la comunidad y destacar para los que le interese (igual no tengo pensado que funcione el pago es visual)

Tambien va a ver una parte de comunidad para poder compartir experiencias,tips,etc



------- FUNCIONES -------

El logo te lleva a la HOME

El menu esta funcional luego se puede mejorar con JS

El scroll to top esta funcional cuando veamos JS se va a incluir que aparezca luego de hacer scroll y que no este desde el principio

Comunidad, Postularse y Ranking estan con comming soon



------- DESCRIPCION DE PAGINAS -------
